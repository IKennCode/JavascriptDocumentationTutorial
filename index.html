KEYWORD DESCRIPTION
var Declares a variable
let Declares a block variable
const Declares a block variable
if Marks a block of statements to be executed on condition
switch Marks a block statements to be executed in different cases
for Marks a block statements to be executed in a loop
function Declares a function
return Exits a function
try Implements a error handling to a block statements
